# Featured Project

<div align="center">

## STRUMM

### *Where Your Music Lives.*

<img src="https://github.com/Agnibha007/Portfolio-New/blob/main/public/images/project11.png" width="100%"/>

</div>

A modern music platform built around personalization rather than playlists.

Instead of just streaming music, Strumm focuses on helping users rediscover songs through intelligent recommendations, listening analytics, and a clean, distraction-free experience.

### Highlights

- Personalized recommendations
- Replay analytics
- Sound DNA
- Smart playlists
- Lyrics support
- Progressive Web App
- Responsive playback
- Modern UI built with Next.js
